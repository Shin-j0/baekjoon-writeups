## 입력


첫째 줄에 n이 주어진다. (1 ≤ n ≤ 1,000)


## 출력


첫째 줄에 2×n 크기의 직사각형을 채우는 방법의 수를 10,007로 나눈 나머지를 출력한다.


## 예제 입력 


```plain text
8
```


## 예제 출력 


```plain text
171
```

dp[1] = 1, dp[2]=3, dp[3]=5 . . . dp[n] = dp[n-2]*2 + dp[n-1]
