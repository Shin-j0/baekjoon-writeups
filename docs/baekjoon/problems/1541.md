## 문제


세준이는 양수와 +, -, 그리고 괄호를 가지고 식을 만들었다. 그리고 나서 세준이는 괄호를 모두 지웠다.


그리고 나서 세준이는 괄호를 적절히 쳐서 이 식의 값을 최소로 만들려고 한다.


괄호를 적절히 쳐서 이 식의 값을 최소로 만드는 프로그램을 작성하시오.


## 입력


첫째 줄에 식이 주어진다. 식은 ‘0’~‘9’, ‘+’, 그리고 ‘-’만으로 이루어져 있고, 가장 처음과 마지막 문자는 숫자이다. 그리고 연속해서 두 개 이상의 연산자가 나타나지 않고, 5자리보다 많이 연속되는 숫자는 없다. 수는 0으로 시작할 수 있다. 입력으로 주어지는 식의 길이는 50보다 작거나 같다.


## 출력


첫째 줄에 정답을 출력한다.


## 예제 입력


```plain text
55-50+40
```


## 예제 출력


```plain text
-35
```


## 해결 방안


주어신 수식에서 원하는 위치에 괄호를 추가하여 최소 값을 찾는 문제. 


괄호와 - 가 만나면 이후에 나오는 + 연산은 모두 - 로 치환 가능 → -(10+20) = -30


해당 원리를 이용하면 첫 숫자부터 - 가 나올 때 까지의 수는 유일한 양수가 된다.


- 이후에 나오는 + 연산은 모두 진행, - 가 다시 나오면 연산을 중단하고 리셋 하여 다시 시작.


50 - 10 +20 - 20 +30 → 50 - (10+20) -(20+30) = -30


결국 - 기준으로 split하고 나눠진 수식 중 처음 수식의 값에서 나머지 모든 수식의 값을 빼면 된다.


## 알고리즘

1. 수식을 - 기준으로 split하여 vector에 저장.
2. 이후 연산자가 나올 때 마다 stoi를 통해 문자열 값을 정수로 변경하여 수식의 값을 추출
3. 첫 수식의 값에서 나머지 수식의 값을 뺀 후 값 출력.
