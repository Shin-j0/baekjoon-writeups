## 문제


N+1개의 `I`와 N개의 `O`로 이루어져 있으면, `I`와 `O`이 교대로 나오는 문자열을 PN이라고 한다.

- P1 `IOI`
- P2 `IOIOI`
- P3 `IOIOIOI`
- Pn `IOIOI...OI` (`O`가 N개)

`I`와 `O`로만 이루어진 문자열 S와 정수 N이 주어졌을 때, S안에 PN이 몇 군데 포함되어 있는지 구하는 프로그램을 작성하시오.


## 입력


첫째 줄에 N이 주어진다. 둘째 줄에는 S의 길이 M이 주어지며, 셋째 줄에 S가 주어진다.


## 출력


S에 PN이 몇 군데 포함되어 있는지 출력한다.


## 제한

- 1 ≤ N ≤ 1,000,000
- 2N+1 ≤ M ≤ 1,000,000
- S는 `I`와 `O`로만 이루어져 있다.

## 서브태스크


| 번호 | 배점 | 제한                   |
| -- | -- | -------------------- |
| 1  | 50 | N ≤ 100, M ≤ 10 000. |
| 2  | 50 | 추가적인 제약 조건이 없다.      |


## 예제 입력 


```plain text
1
13
OOIOIOIOIIOII
```


## 예제 출력 


```plain text
4
```


## 해결 방안


주어진 문자열 속에 ioi가 몇 개 들어있는지 찾으면 되는 문제


while(pos) pos ← str.find(ioi) 이런 식으로 작성하면 50점만 나온다. 시간 복잡도를 해결하기 위해


I O I → i-1, i , i+1 if문으로 체크하여 참이면 c+=1 / i+=2, 거짓이면 c= 0 / i+=1


이렇게 해서  c ≥ n 일 때 마다 카운트 한다.


c≥n인 이유는 n = ioioi(2)  str = ioioioioi 일때 


| index  |  0 | 1 | 2 | 3 | 4 | 5 | 6 | 7 | 8 |
| ------ | -- | - | - | - | - | - | - | - | - |

이런 식으로 처음 ioioi와 겹치는 ioioioi 경우도 생각해야 하기 때문이다.
