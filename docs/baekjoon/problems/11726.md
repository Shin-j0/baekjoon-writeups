## 문제


2×n 크기의 직사각형을 1×2, 2×1 타일로 채우는 방법의 수를 구하는 프로그램을 작성하시오.


아래 그림은 2×5 크기의 직사각형을 채운 한 가지 방법의 예이다.


![1.png](https://onlinejudgeimages.s3-ap-northeast-1.amazonaws.com/problem/11726/1.png)


## 입력


첫째 줄에 n이 주어진다. (1 ≤ n ≤ 1,000)


## 출력


첫째 줄에 2×n 크기의 직사각형을 채우는 방법의 수를 10,007로 나눈 나머지를 출력한다.


## 예제 입력 


```plain text
9
```


## 예제 출력 


```plain text
55
```


## 해결 방안


2xn = ? 


2x1 =  1 


2x2 = 11 2


2x3 = 111 21 12


2x4 = 1111 211 121 112 22


2x5 = 11111 2111 1211 1121 1112 221 212 122


n = (n-1) + (n-2) 처럼 값이 증가함

## 알고리즘

1. 다이나믹 프로그래밍 문제는 규칙/조건이 주어짐 → 해당 문제는 dp 피보나치와 비슷한 유형
2. dp  피보나치와 비슷하게 dp[i] = dp[i-1] +dp[i-2] 로 요소의 값 구하기
3. 이때 dp[i] 마다 10007을 해주는 이유 → 피보나치의 값은 기하 급수적으로 커지기에

    long long 을 사용해도 오버  플로우가 발생한다. 


    n=1000일 때 의 값 - 209자리 수


    unsigned long long의 값은 10^19 즉 19자리 수 까지만 얻을 수 있다. 


    → 큰 값을 계산 시 mod를 통해 나머지 값만 비교하는 방식을 Modular Arithmetic 이라고 한다
